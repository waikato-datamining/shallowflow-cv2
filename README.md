# shallowflow-cv2
Imaging components using [OpenCV2](https://github.com/opencv/opencv-python) for shallowflow.

## Installation

Install via pip:

```bash
pip install "git+https://github.com/waikato-datamining/shallowflow-cv2.git"
```
